2. Überblick

TraceTour ist eine browserbasierte Web-App. Wir nehmen den Schutz deiner persönlichen Daten ernst. Diese Datenschutzerklärung informiert dich darüber, welche Daten erhoben werden und wie wir damit umgehen.

3. Hosting

Die Website wird über GitHub Pages (GitHub Inc., San Francisco, USA) gehostet. GitHub kann dabei Server-Logdaten erheben (IP-Adresse, Browsertyp, Zeitpunkt des Zugriffs). Weitere Informationen findest du in der Datenschutzerklärung von GitHub.

4. Lokale Datenspeicherung

TraceTour speichert deinen Spielfortschritt ausschließlich lokal in deinem Browser (localStorage). Folgende Daten werden gespeichert:

  • Dein eingegebener Name (für die Urkunde)
  • Gelöste Stationen und gesammelte Punkte
  • Aktuelle Position in der Tour

Diese Daten werden nicht an einen Server übertragen. Du kannst sie jederzeit über die App-Funktion „Fortschritt löschen" oder über die Browser-Einstellungen entfernen.

5. Standortdaten

Die App fragt über deinen Browser nach deinem Standort (GPS), um deine Position auf der Karte anzuzeigen und die Entfernung zur nächsten Station zu berechnen. Die Standortdaten werden ausschließlich lokal in deinem Browser verarbeitet und nicht an unsere Server oder Dritte übertragen. Du kannst die Standortfreigabe jederzeit in deinen Browser-Einstellungen widerrufen.

6. Trace (digitaler Begleiter)

Der optionale Begleiter „Trace" sendet deine Frage an einen Cloudflare Worker, der die Anfrage an die Anthropic API weiterleitet. Dabei werden übertragen:

  • Dein Fragetext (max. 500 Zeichen)
  • Kontext zur aktuellen Station (Rätsel, Hinweise)

Es werden keine personenbezogenen Daten (Name, Standort, E-Mail) an den Dienst übertragen. Die Anfragen werden nicht gespeichert. Rate-Limiting erfolgt über die IP-Adresse, die im Cloudflare Worker verarbeitet, aber nicht dauerhaft gespeichert wird.

Weitere Informationen: Anthropic Privacy Policy

7. Externe Dienste

Die App nutzt folgende externe Dienste:

  • OpenStreetMap / Leaflet – für die Kartenanzeige. Kartenkacheln werden von OpenStreetMap-Servern geladen. OSM Datenschutz
  • Google Fonts – für Schriftarten (nur auf der Landingpage). Google Datenschutz
  • Google Maps – für Navigationslinks (werden in einem neuen Tab geöffnet)

8. Cookies

TraceTour selbst setzt keine Cookies. Externe Dienste (GitHub Pages, OpenStreetMap) können eigene Cookies setzen.

9. Audiodateien

Alle Audiodateien werden direkt von unserem Server geladen. Es findet kein Streaming über Drittanbieter statt.
